# Wine Quality Prediction

This project predicts whether a wine is of good quality based on its chemical properties using machine learning.

## Objective
To classify wine as:
- Good quality (quality >= 7)
- Not good quality (quality < 7)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Models Used
- Logistic Regression
- Random Forest Classifier

## Workflow
1. Load and inspect the dataset
2. Explore the quality distribution
3. Create a binary target
4. Split into training and test sets
5. Train classification models
6. Evaluate performance
7. Analyse feature importance

## Key Insight
Random Forest generally performs better than Logistic Regression for this tabular classification problem.
